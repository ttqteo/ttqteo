# Keyboard Shortcuts - Phím Tắt

## Tổng Quan

Mindmap hỗ trợ nhiều phím tắt để thao tác nhanh.

---

## 1. Điều Hướng (Navigation)

| Phím tắt  | Mô tả                               |
| --------- | ----------------------------------- |
| `Alt + ↑` | Di chuyển đến node anh em phía trên |
| `Alt + ↓` | Di chuyển đến node anh em phía dưới |
| `Alt + ←` | Di chuyển đến node cha              |
| `Alt + →` | Di chuyển đến node con đầu tiên     |

---

## 2. Chỉnh Sửa (Editing)

| Phím tắt               | Mô tả                      |
| ---------------------- | -------------------------- |
| `Click`                | Chọn và chỉnh sửa node     |
| `Enter`                | Lưu và tạo node anh em mới |
| `Shift + Enter`        | Xuống dòng trong node      |
| `Tab`                  | Lưu và tạo node con mới    |
| `Shift + Tab`          | Thăng cấp node (promote)   |
| `Escape`               | Hủy chỉnh sửa              |
| `Delete` / `Backspace` | Xóa node (khi text rỗng)   |

---

## 3. Note (Ghi Chú)

| Phím tắt      | Mô tả                   |
| ------------- | ----------------------- |
| `Alt + N`     | Focus vào ô nhập Note   |
| `Alt + T`     | Quay lại ô nhập Title   |
| `Alt + Enter` | Lưu note và đóng editor |

---

## 4. Undo/Redo (Hoàn Tác)

| Phím tắt           | Mô tả                               |
| ------------------ | ----------------------------------- |
| `Ctrl + Z`         | Undo - Hoàn tác thao tác trước      |
| `Ctrl + Y`         | Redo - Làm lại thao tác đã hoàn tác |
| `Ctrl + Shift + Z` | Redo (cách khác)                    |

> **Lưu ý:** Undo/Redo lưu tối đa 50 trạng thái.

---

## 5. Zoom & Pan

| Phím tắt         | Mô tả                         |
| ---------------- | ----------------------------- |
| `Ctrl + Scroll`  | Zoom in/out                   |
| `Shift + Scroll` | Pan ngang (horizontal scroll) |
| `Drag`           | Kéo để di chuyển canvas       |

---

## 6. Radial Context Menu

| Thao tác      | Mô tả                         |
| ------------- | ----------------------------- |
| `Right-click` | Mở menu tròn với các tùy chọn |

### Các tùy chọn trong menu:

- **Reset View** - Đặt lại zoom về 100% và vị trí gốc
- **Add Root** - Thêm root node mới
- **Center View** - Căn giữa toàn bộ mindmap
- **Zoom Out** - Thu nhỏ
- **Zoom In** - Phóng to

---

## 7. Các Thao Tác Khác

| Thao tác        | Mô tả                                            |
| --------------- | ------------------------------------------------ |
| `Click •`       | Click vào chấm tròn cạnh node để thu gọn/mở rộng |
| `Click outside` | Click bên ngoài node để lưu và đóng editor       |
