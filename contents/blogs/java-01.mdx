```java
import java.util.Scanner;

public class HelloWorld {

    public static void main(String[] args) {

        // Creates a reader instance which takes
        // input from standard input - keyboard
        Scanner reader = new Scanner(System.in);
        System.out.print("Enter a number: ");

        // nextInt() reads the next integer from the keyboard
        int number = reader.nextInt();

        // println() prints the following line to the output screen
        System.out.println("You entered: " + number);
    }
}
```

# Java Collections

Set trong java là gì?

- HashSet 
- Độ phức tạp O(1)
```java
HashSet<Integer> set = new HashSet<>();
set.add(1);
set.add(2);
set.add(3);

System.out.println(set);
set.remove(1);
set.contains(2);
set.size();
set.retainAll(set2);
```

- LinkedHashSet
- Độ phức tạp O(1)
```java
LinkedHashSet set = new LinkedHashSet<>();
set.add(1);
set.add(2);
set.add(3);

System.out.println(set);
```

-> LinkedHashSet và HashSet không khác gì nhau lắm

- TreeSet
- Khác biệt: Có thứ tự
- Độ phức tạp O(log(n))
```java
TreeSet set = new TreeSet<>();
set.add(1);
set.add(2);
set.add(3);
```

Vì có thứ tự nên dùng được
```java
TreeSet<Integer> set = new TreeSet<>();
set.first(); // phần tử đầu tiên
set.last(); // phần tử cuối cùng
set.lower(2); // phần tử nhỏ hơn 2
set.higher(2); // phần tử lớn hơn 2
set.ceiling(2); // phần tử nhỏ nhất và lớn hơn hoặc bằng 2
set.floor(2); // phần tử lớn nhất và bé hơn hoặc bằng 2
set.pollFirst(); // xoá phần tử đầu tiên
set.pollLast(); // xoá phần tử cuối cùng
```
