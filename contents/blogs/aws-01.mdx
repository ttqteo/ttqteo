# EC2 - vì sao cần thiết?

- **Elastic Computing Cloud** là dịch vụ máy ảo của AWS
- Tại sao dùng EC2 thay vì Digital Ocean?
+ Viết blog -> nên dùng VPS hoặc Digital, bởi vì nếu lượng truy cập cao -> phí sử dụng EC2 tăng cao
- EC2 có gì nổi bật?
  - Điều chỉnh tỉ lệ, điều chỉnh lưu lượng
  - Dùng bao nhiêu trả bấy nhiêu, tính co giãn
  - AWS Auto scaling
  - Miễn phí 12 tháng 
  - **Độ tin cậy**
  - **Độ co giãn**
- Thực hành
  - Đăng ký EC2
  - AMI (Amazon Machine Image)
  - Instance type
  - Amazon EBS (Elastic Block Storage)
  - Elastic IP address
  - VPCs (Virtual Private Cloud)
